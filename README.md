A Twitter module for use with [ZF2](http://framework.zend.com/).

Use [Composer](https://getcomposer.org/) to install this library. Edit your `composer.json` like so:
```json
{
    "require": {
        "hrphp/hrphp-twitter": "*"
    }
}
```
